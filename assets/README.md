# Portfolio & Achievement Screenshots

Place your project and achievement screenshots in this directory to automatically display them on the website.

## Filenames Used in `index.html`

Ensure your files match these exact names (all lowercase with underscores) and formats:

### 1. Projects
- **Oulu Botanical Garden Navigation System**:
  - Filename: `botani_nav.png`
- **Datamend**:
  - Filename: `datamend.png`
- **CampaignClickR**:
  - Filename: `campaign_clickr.png`
- **BookRAG**:
  - Filename: `bookrag.png`

### 2. Achievements
- **JunctionOulu Hackathon 2025**:
  - Filename: `junction_oulu.png`
- **Junction 2025 (P&G's Challenge)**:
  - Filename: `junction_helsinki.png`

*(Note: If you use other formats like `.jpg`, `.svg`, or `.webp`, simply open `index.html` in a text editor and change the file extensions in the corresponding `src` attributes.)*

## How the Fallback Works

If no image is found in this directory, the website will automatically display a beautiful CSS-styled glassmorphic placeholder card with a relevant icon (e.g. map pins, charts, workflow arrows, trophies). Once you drop a file in (e.g. `junction_oulu.png`), it will instantly replace the placeholder on reload.
